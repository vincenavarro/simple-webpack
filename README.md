# Simple Webpack

A webpack base with comments and limited additional clutter.

## Features

* ES6+ Babel
* HTML Templates
* Prettier Formatting
* SASS + Autoprefixing
* Angry red linter complaints

## Use

* Develop: `yarn dev`
* Build: `yarn build`
* Prettier: `yarn format`
